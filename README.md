# Minimal BB Forum

Lightweight bulletin board forum software built with PHP.

## Features

- User authentication and sessions
- Post creation and reply functionality
- Thread-based discussion system
- Simple, clean interface
- Minimal dependencies

## Structure

```
src/
├── models/      - Data models (Post, User)
├── routes/      - Request routing
└── views/       - UI templates
```

## Installation

1. Configure your web server to point to this directory
2. Set up database connection
3. Import database schema
4. Access `index.php` in your browser

## Requirements

- PHP 7.0 or higher
- MySQL/MariaDB database
- Apache/Nginx web server

## Usage

The forum provides basic bulletin board functionality:
- Create new discussion threads
- Reply to existing threads
- User registration and login
- View post history
